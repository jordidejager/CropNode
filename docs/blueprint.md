# **App Name**: AgriSprayer Pro

## Core Features:

- Natural Language Spray Input: Users can input spray application details in natural language or text.
- AI-Powered Data Extraction: Use Gemini AI to parse the input, identify plots, products, and dosages, acting as a tool to extract structured information.
- Regulatory Compliance Validation: Validate the application against the 'MiddelMatrix' to check legality and dosage limits, providing warnings if needed.
- Logbook Recording: Store spray application data in a human-readable logbook format for easy review.
- Parcel History Tracking: Split application data into individual parcel entries to track spray history per parcel.
- Interactive Dashboard & Filters: Provide an interactive dashboard with filters for crop, parcel, and date to review application history.
- Database Management: Utilize unique IDs for parcels to allow easy parcel name changes without breaking history and automatically add new products to the database when used.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to reflect the agricultural nature of the app.
- Background color: Light beige (#F5F5DC), providing a neutral and calming backdrop.
- Accent color: Muted gold (#BDB76B) to highlight important information and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern and slightly warm feel.
- Use clear and intuitive icons representing crops, products, and application types.
- Design a clean and organized layout with easy navigation between the five core tabs (Input, Logbook, ParcelHistory, Parcels, MiddelMatrix).
- Use subtle transitions and animations to provide feedback and guide the user through the application process.
